# Low Poly Coop (lowpolycoop.org)

[![License: CC BY 2.5](https://img.shields.io/badge/License-CC%20BY%202.5-lightgrey.svg)](https://creativecommons.org/licenses/by/2.5/)

A legacy collection of low-poly 3D models designed for rapid game prototyping. Originally established around 2006, this project served as a community-driven resource for game developers and 3D artists across various forums.

## 🏛 Project History

The **Low Poly Coop** was born from the mid-2000s indie game development scene. During an era where "low poly" was a technical necessity rather than just an aesthetic choice, artists and developers from across the web collaborated to build a shared library of optimized assets. Many of these models were originally shared on game development and digital art forums, helping countless developers get their projects off the ground.

Today, this repository serves as a modern home for these heritage assets, ensuring they remain accessible for a new generation of developers.

## 🛠 Technical Specifications

To ensure compatibility and ease of use, all models follow these standards:

- **Formats:** Primary source files are in **Blender (`.blend`)** format.
- **Textures:** Optimized **512x512** diffuse maps.
- **UV Maps:** Standardized **1024x1024** UV templates.
- **Optimization:** Models are designed to be extremely lightweight, focusing on low triangle counts suitable for mobile, web, and retro-style games.

## 🤝 Contributing

We welcome new contributions that fit the project's low-poly aesthetic and technical constraints. If you would like to contribute:

2.  Use the [model_info.md](templates/model_info.md) template for your model's documentation.
3.  Ensure all work submitted is your own original creation.
